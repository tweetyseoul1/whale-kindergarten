■ 웨일어린이집 홈페이지 — 사용 안내

【파일 구성】
  whale-kindergarten.html  : 홈페이지 (더블클릭하면 열립니다)
  admin.html               : 관리자 페이지 (이미지 교체)
  images 폴더              : 사진 폴더 (아래 이름으로 넣으면 자동 표시)
  serve-whale.ps1          : 로컬 서버 실행 스크립트 (선택사항)

【홈페이지 열기】
  whale-kindergarten.html 파일을 더블클릭하면 바로 열립니다.

【이미지 넣는 방법 2가지】
  방법 1) admin.html(관리자 페이지)에서 [이미지 올리기]
          → 그 컴퓨터·브라우저에서 즉시 반영
  방법 2) images 폴더에 아래 이름으로 사진 파일을 넣기
          → 어디서 열어도 항상 표시 (추천)

     images/hero-kids.jpg      메인 대표 사진 (첫 화면 전체)
     images/prog-language.jpg  언어놀이
     images/prog-art.jpg       미술활동
     images/prog-outdoor.jpg   바깥놀이
     images/prog-sensory.jpg   감각놀이
     images/facility.jpg       원 내부 시설

     (jpg 외에 png, webp, jpeg 도 가능합니다)

【다른 노트북에서 클로드(Claude)로 계속 작업하기】
  1. 이 ZIP을 새 노트북에 복사해서 압축을 풉니다.
  2. 클로드 앱에서 압축 푼 폴더를 작업 폴더로 열고
     "이 폴더의 웨일어린이집 홈페이지를 수정해줘" 라고 말하면 됩니다.
